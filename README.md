# Programmentwurf TIT20 2022
Es handelt sich um ein Programm, welches eine Liste von Zeitstemepel einliest und den Zeitunterschied zwischen Zeitstempel ausgibt.

## usage
Normal Usage:
```
cd pe2022
make
cat eingabe.txt | ./timediff
```

List Unit Test:
```
cd pe2022
make
./list_test
```

### dependencies
 - nasm
 - make
